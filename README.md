## 🚀 Inicio Rápido

### Prerequisitos

- Node.js 18+
- pnpm

### Instalación

1. **Clona el repositorio**

   ```bash
   git clone https://github.com/tuusuario/portfolio.git
   cd portfolio
   ```

2. **Instala las dependencias**

   ```bash
   pnpm install
   ```

3. **Inicia el servidor de desarrollo**

   ```bash
   pnpm run dev
   ```

4. **Abre en tu navegador**
   ```
   http://localhost:3000
   ```
